# Basic Calculator

A calculator project built with HTML, CSS and JavaScript as part of my programming learning journey.

### Features
- Basic math operations (+, -, *, /);
- Decimal number support;
- Keyboard support;
- Negative number support;
- Dark mode UI;
- CSS Grid layout;
- Custom font styling;
- Floating-point precision fixes.

### Technologies
- HTML5;
- CSS3;
- JavaScript.
